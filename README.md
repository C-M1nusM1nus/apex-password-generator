# 🔐 Apex Password Generator

Apex Password Generator is a modern, responsive web application for creating secure, customizable passwords. Built with clean UI principles and interactive feedback, it demonstrates strong front-end development skills and attention to user experience.

---

## 🚀 Features

* 🎯 Generate random passwords instantly
* 📏 Adjustable password length (6–32 characters)
* 🔡 Custom character selection:

  * Lowercase letters
  * Uppercase letters
  * Numbers
  * Special characters
* 💪 Real-time password strength indicator
* 📋 One-click copy to clipboard
* 📱 Fully responsive (mobile + desktop)
* 🎨 Modern, polished UI with smooth interactions

## 📦 Installation & Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/apex-password-generator.git
   ```

2. Navigate into the project folder:

   ```bash
   cd apex-password-generator
   ```

3. Open `index.html` in your browser.

---

## 🌐 Deployment (GitHub Pages)

1. Go to your repository settings
2. Navigate to **Pages**
3. Under *Source*, select:

   * Branch: `main`
   * Folder: `/root`
4. Save and wait a few seconds

Your app will be live at:

```
https://your-username.github.io/apex-password-generator/
```
